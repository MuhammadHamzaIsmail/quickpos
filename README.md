# ⚡ QuickPOS - Point of Sale Landing Page

## Setup Instructions

1. Install MAMP/XAMPP
2. Start Apache server
3. Copy folder to htdocs
4. Open http://localhost/QuickPOS/index.php

## Features
- Responsive navbar with logo and navigation
- Hero section with CTA buttons and dashboard mockup
- 4 feature cards with icons
- 3-tier pricing table (Basic $0, Pro $29, Enterprise $99)
- Contact form with PHP validation
- Footer with social links

## Technology
- PHP 8
- HTML5/CSS3
- Google Fonts (Inter)
